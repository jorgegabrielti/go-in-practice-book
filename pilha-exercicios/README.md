# 🏋️ Pilha de Exercícios — Prática Deliberada

Inspirado na [lista de exercícios do dunossauro](https://exercicios.dunossauro.com/), esta pasta reúne **enunciados extras** (sem solução pronta) para fixação de cada capítulo, além dos exercícios oficiais já resolvidos em `estudos-go/capituloXX/exercicios/`.

## Por que uma pasta separada

Os exercícios em `estudos-go/capituloXX/exercicios/` são os propostos pelo próprio livro e já vêm com a resolução no repositório. Esta pilha é diferente: são problemas adicionais, autorais, **propositalmente sem resposta**, para praticar de verdade (você só aprende fazendo, não lendo a solução).

## Regras de cada lista

- **Mínimo de 10 exercícios por capítulo.**
- **Dificuldade crescente**: os primeiros são de aquecimento, os últimos exigem combinar vários conceitos.
- **Cumulativo**: a lista do capítulo N pode usar qualquer conceito ensinado nos capítulos 1 a N (não só do capítulo atual) — assim como nas listas do dunossauro, cada exercício novo pode reaproveitar o que já foi aprendido antes.
- **Sem solução no arquivo de enunciados.** Se quiser resolver, crie uma pasta própria (ex: `estudos-go/capituloXX/exercicios/practice01/`) ou um scratch separado — o objetivo é o esforço de recuperação (*retrieval practice*), não copiar uma resposta.

## Legenda de dificuldade

🟢 Fácil · 🟡 Médio · 🔴 Desafio

## Como usar

1. Termine o capítulo em `estudos-go/capituloXX/` (teoria + exercícios oficiais).
2. Copie `_template.md` para `capituloXX.md` e escreva pelo menos 10 enunciados, do mais fácil ao mais difícil, baseando-se no que já foi ensinado até este capítulo (consulte `estudos-go/GLOSSARIO.md` se precisar relembrar conceitos antigos).
3. Resolva os exercícios por conta própria, sem pressa — não há gabarito aqui de propósito.
