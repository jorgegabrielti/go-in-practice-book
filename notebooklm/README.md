# 🧠 NotebookLM — Fontes de Estudo por Capítulo

Esta pasta contém arquivos de referência ricos destinados a alimentar o **Google NotebookLM** como professor particular de Go.

## Por que este material existe

O livro *Go na Prática* é excelente: conciso, direto, cheio de boas analogias. Mas justamente por ser conciso, alguns conceitos ficam com pouca profundidade. Estes arquivos existem para resolver isso:

- Expandem cada conceito com explicações que o livro não cobre
- Conectam os tópicos às **referências oficiais da linguagem Go** (go.dev, spec, blog)
- Incluem **Q&A** antecipando as dúvidas mais frequentes
- Comparam o Go com Python, Java e JavaScript para quem vem de outro background
- Listam armadilhas comuns e como evitá-las

## Como usar com o NotebookLM

1. Acesse [notebooklm.google.com](https://notebooklm.google.com)
2. Crie um notebook por capítulo (ou um único com todos)
3. Faça upload do arquivo `capituloXX.md` correspondente
4. Faça perguntas ao NotebookLM como faria a um professor:
   - "Me explique por que o Go usa zero value em vez de null"
   - "Qual a diferença entre make e new na prática?"
   - "Por que Pointer Receiver é preferido na maioria dos casos?"
